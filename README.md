## 🛠 שלב 1: הבנת תנאים (If-Statements)

הבסיס לכל תוכנה חכמה הוא היכולת לקבל החלטות. בתרגיל הראשון, בנינו **מחשבון מס הכנסה** בעברית.
התלמידים למדו להשתמש בלווגיקה של `if`, `elif` ו-`else` כדי לחשב את גובה המס לפי רמות שכר שונות.

**הלוגיקה שיושמה:**

* שכר מתחת ל-6,000 ₪ ⬅️ 10% מס.
* שכר בין 6,000 ל-15,000 ₪ ⬅️ 15% מס.
* שכר מעל 15,000 ₪ ⬅️ 20% מס.

```python
# קוד המקור בעברית
if salary < 6000:
    tax_rate = 0.10
elif salary < 15000:
    tax_rate = 0.15

```

## 🤖 שלב 2: שדרוג הקוד באמצעות AI

לאחר כתיבת הלוגיקה הבסיסית, השתמשנו בכלי **AI** (כמו Gemini/ChatGPT) כדי לשפר את הפרויקט.
בתהליך זה התלמידים למדו כיצד "לדבר" עם ה-AI (Prompt Engineering) כדי לבצע:

1. **לוקליזציה (Localization):** תרגום אוטומטי של כל הממשק לאנגלית כדי להגיע לקהל בינלאומי.
2. **אופטימיזציה של שמות משתנים:** שינוי שמות משתנים מעברית לאנגלית תקנית (לדוגמה: `salary` במקום `שכר`).
3. **פתרון בעיות טכניות:** זיהוי שגיאות קידוד (Encoding) שגרמו לעברית להופיע כסימני שאלה בטרמינל, ומעבר לעבודה ב-Terminal האמיתי שתומך ב-UTF-8.
